## 🔴 Phase 5 — Workout Tracking

**Status: Not started.** Dependency: Phase 1 done ✅

### Pre-step — Fetch Exercise Library

Run one-time script to pull from wger public API and save to `datasets/exercise_library.json`:

```
GET https://wger.de/api/v2/exerciseinfo/?format=json&language=2&limit=100
(paginate all pages)
```

Extract: `name`, `description` (strip HTML), `category.name` → our category + MET value, `muscles[*].name_en`, `equipment[*].name`, `aliases`.

MET values by wger category: Cardio=7.0, Strength=3.5, Stretching=2.3, Yoga=3.0, Plyometrics=8.0, Body part categories (Shoulders/Arms/etc)=3.5.
